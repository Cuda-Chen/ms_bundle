## ms_bundle
Bundle for manipulate with miniSEED data.

## Clone
```
$ git clone --recurse-submodules https://github.com/Cuda-Chen/ms_bundle.git
```

## Compile and Install
```
$ cd src
$ make download
$ make
$ make install
```

## Where're the Built Programs?
All of the built programs are in `bin` directory.
